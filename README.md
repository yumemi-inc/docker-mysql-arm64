# docker-mysql-multiarch

> **Warning**
> This is not an official product of YUMEMI Inc.

MySQL Docker Image with Arm64 architecture support for 5.7.

## Usage

### From CLI

```shell
docker run -d ghcr.io/yumemi-inc/mysql:5.7
```

### From Dockerfile

```dockerfile
FROM ghcr.io/yumemi-inc/mysql:5.7
```
